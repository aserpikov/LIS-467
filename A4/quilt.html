Learn to Quilt

Beginner Quilting

This video series from Melanie Ham walks beginners through everything they will need to know to get started quilting, and to make their first quilt! Melanie also has a blog that walks through the content in the above video and provides plenty of other articles and resources to get staretd with quilting.

Learning Resources:

  • Learn the Basics of Quilting from The Seasoned Homemaker
  • How to make a Quilt: Project for Beginners from We All Sew

Hand Quilting

This video from Suzy Quilts walks beginners through the basics of hand quilting, and covers related topics such as how to pick thread color, knotting your thread, sewing a running stitch, and much more! Hand quilting is useful for any quilter, as some patterns may have smaller or more complex pieces that would be easier to finish off by hand.

Learning Resources:

  • Hand-Quilting Basics from All People Quilt
  • 6 Techniques for Hand Quilting from Martha Stewart

Patterns

Take a look at some patterns sorted by difficulty below:

Zig Zag Quilt Pattern

This is an easy zig zag quilting pattern & tutorial that's perfect for beginners to try out quilting. The tutorial uses pre-cut pieces and charm packs, but you could easily cut your own quilitng pieces.

Link: Zig Zag Quilt Pattern

Herringbone Quilt Pattern

This tutorial provides a pattern and tutorial for an easy herringbone quilt. The tutorial provides measurements for different sized blankets (Twin, Full, Queen, King, etc.) which makes it easy for beginners to pick out the right size for them, or to start on a smaller measurement (ex. Baby/Crib size) to build up skill.

Link: Herringbone Quilt Pattern

Chevron Quilt Pattern

This beginner chevron quilt tutorial provides a printable pattern template and two sizes (lap & twin sizes). This pattern also only uses 10-inch squares (one pattern piece) for the entire project, making it a great beginner level quilt.

Link: Chevron Quilt Pattern

Harmony Square Quilt Block

This project includes a printable pattern and step-by-step instructions on how to assemble the block with pictures. The tutorial uses a lot of acronyms for quilting terms that you may want to be familiar with before starting, making this a more intermediate level pattern.

Link: Harmony Square Quilt Block Pattern

Spring Quilt

This Spring Quilt pattern works with smaller pattern pieces in a less traditional quilting pattern. The project is straightforward but requires patience and attention to detail, making a good test of your skills.

Link: Spring Quilt Pattern

Scrappy Mountain Quilt

This project works of off triangle templates which can be purchased from the creators Etsy shop, but can easily be recreated by hand, especially since the creator includes measurements for all the pattern pieces directly in the tutorial!

Link: Scrappy Mountain Quilt Pattern

Snail's Trail Quilt Block

This quilt block is a bit more advanced since it includes a spiral pattern that can be tricky to construct, but presents a fun challenge to experienced quilters.

Link: Snail's Trail Quilt Block Pattern

Yo-Yo Quilt

This quilt isn't particularly difficult in construction, but it requires you to make individual Yo-yos (also known as rosettes) to quilt together, making this project a bit more involved than when using flat pattern pieces.

Link: Yo-Yo Quilt Pattern
